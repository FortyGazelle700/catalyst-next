# 0.5.2

## TL;DR

First "public" (private beta, with people other than developers) release, expect lots of bugs and issue with the first build.

## Release Notes

This is the first "public" release of Catalyst. This version is a private alpha release of Catalyst released to a small group of testers allowing developers to get necessary data on user interaction and feedback. This first release in a rough stage, however, in a stage where it is usable.

## Known Issues

- File Submission do not work when using the published site (works internally)
- Many pages are marked as "Under Construction"
- Some "Missing Assignments" may be misclassified
- Mobile versions are a little clunky, rendering issues will occur
- Grade calculator may produce inaccurate results
- Changes may not appear immediately after update
  - After submission
  - On edit

## Planned Features

1. Fix major bugs and issues
2. Fix Known Issues
3. Addition of School Tab
   1. Add schedules to specific dates via Catalyst UI
   2. Show all schedules, show upcoming schedules
